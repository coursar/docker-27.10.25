# Установка Docker

**Важно**:
- логин: `student`
- пароль: `pass`

## Шаг 1. Установка Docker Engine

Согласно [руководству по установке](https://docs.docker.com/engine/install/ubuntu) выполните установку Docker Engine

1. Удаление старых пакетов:

```sh
for pkg in docker.io docker-doc docker-compose docker-compose-v2 podman-docker containerd runc; do sudo apt-get remove $pkg; done
```

2. Добавление репозиториев:

```sh
# Add Docker's official GPG key:
sudo apt-get update
sudo apt-get install ca-certificates curl
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
sudo chmod a+r /etc/apt/keyrings/docker.asc

# Add the repository to Apt sources:
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "${UBUNTU_CODENAME:-$VERSION_CODENAME}") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
sudo apt-get update
```

3. Установка пакетов:

```sh
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

4. Проверка статуса сервиса:

```sh
sudo systemctl status docker
```

5. Запуск контейнера из образа [hello-world](https://hub.docker.com/_/hello-world):

```sh
sudo docker run hello-world
```

## Шаг 2. Пост-установочные шаги

Согласно [руководству по пост-установочным шагам](https://docs.docker.com/engine/install/linux-postinstall) добавьте пользователю `student` возможность запускать Docker

1. Добавьте пользователя в группу `docker`:

```sh
sudo usermod -aG docker student
```

2. Перелогиньтесь

3. Запустите `hello-world` без `sudo:

```sh
docker run hello-world
```
