# Dive

## Шаг 1. Установка

Установите [`dockle`](https://github.com/goodwithtech/dockle/releases):
1. Скачайте установочный файл (Linux-64bit.deb)
2. Установите: `sudo apt install ./...Linux-64bit.deb`

## Шаг 2. Анализ образа

Проанализируйте один из созданных ранее образов на уязвимости и секреты:

```sh
dockle IMAGE:TAG
```

Обратите внимание на код возврата (даже в случае ошибок):

```sh
echo $?
```

Так же, как и в trivy, для установки кода возврата необходимо использовать [флаг `--exit-code`](https://github.com/goodwithtech/dockle?tab=readme-ov-file#specify-exit-code)
