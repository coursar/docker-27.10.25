# Nginx

## Шаг 1. Написание yml

Используя образ [nginx](https://hub.docker.com/_/nginx), напишите `docker-compose.yml` для Nginx с учётом следующих требований:

1. Tag: 1.29.3-alpine3.22

2. Port publishing: 8080 хостовой на 80 в контейнере

3. Volume Mapping: htdocs хостовой на /usr/share/nginx/html в контейнере

## Шаг 2. Запуск

Запустите docker compose (запускать надо в том же каталоге, где находится `docker-compose.yml`, либо указывать путь к нему через `-f`)

```sh
docker compose up
```

## Шаг 3. Просмотр приложения

Перейдите в браузере Firefox по адресу http://localhost:8080, скопируйте выведенное на странице значение:

```
...fff4
```

Полученное значение и будет ответом

## Шаг 4. Завершение работы контейнера

Завершите работу поднятых сервисов с помощью `Ctrl + C` в терминале с docker compose
