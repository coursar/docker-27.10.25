# Tomcat WAR

Соберите с помощью Maven проект из каталога tomcat10 в war и разверните в Tomcat 10.1.48

## Шаг 1. Написание Dockerfile

Напишите multistage Dockerfile:
1. Stage builder: сборка с помощью Maven (используйте образ Maven 3 с JDK 17), команда сборки: `mvn -B package`
2. Stage release: разверните получившийся war в Tomcat (используйте образ Tomcat 10.1.48 с JDK 17)*

Пользователя настраивать не нужно

Подсказка*: для развёртывания скопируйте war в файл $CATALINA_HOME/webapps/ROOT.war

Дайте образу тег webapp:1.0-tomcat10

## Шаг 2. Запуск Tomcat

Запустите получившийся образ с публикацией порта 8080, сделайте запрос на http://localhost:8080

Значение в поле value и будет ответом
