# Spring Boot Maven

Соберите с помощью Maven (не Maven Wrapper) проект из каталога boot-web-maven и разверните в образе Eclipse Temurin или [OpenJDK от Oracle](https://container-registry.oracle.com/ords/ocr/ba/java/openjdk)*

Примечание*: возможно, будет медленно качаться

## Шаг 1. Изучение базовых образов

В качестве базовых образов используйте:
1. Maven 3.9 JDK 25 Alpine
2. Eclipse Temurin или [Oracle OpenJDK - 25.0.1-ol9-amd64](https://container-registry.oracle.com/ords/ocr/ba/java/openjdk)

Скачайте образа и с помощью команды `docker inspect -f'{{.Config}}' IMAGE` изучите конфигурацию (в частности, переменные окружения, Cmd и Entrypoint при наличии), подумайте, что вам нужно будет переопределить в ваших образах

## Шаг 2. Написание Dockerfile

Напишите multistage Dockerfile:
1. Stage builder: сборка с помощью Maven , команда сборки: `mvn -B package`
2. Stage release: разверните получившийся jar в образе Eclipse Temurin или Oracle OpenJDK

Пользователя настраивать не нужно

Дайте образу тег webapp:1.0-boot

## Шаг 2. Запуск Tomcat

Запустите получившийся образ с публикацией порта 8080, сделайте запрос на http://localhost:8080/api/info

Значение в поле value и будет ответом
