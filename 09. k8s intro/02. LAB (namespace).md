# K8S Namespaces

## Шаг 0. Подключение

Удостоверьтесь, что можете с помощью `kubectl` получить информацию о кластере:

```sh
kubectl cluster-info
```

## Шаг 1. Создание Namespace

Создайте Namespace командой:

```sh
kubectl apply -f 02_namespace.yml
```

Просмотрите список существующих Namespace:

```sh
kubectl get namespaces
```

Удостоверьтесь, что среди них есть `appns`
