# Hadolint

## Шаг 1. Установка

Установите [`hadolint`](https://github.com/hadolint/hadolint/releases):
1. Скачайте бинарный файл (hadolint-linux-x86_64)
2. Установите в PATH: `sudo install hadolint-linux-x86_64 /usr/local/bin/hadolint`

## Шаг 2. Анализ Dockerfile

Проанализируйте один из `Dockerfile` для созданных образов:

```sh
hadolint -V Dockerfile
```

В случае отсутствия ошибок (код возврата 0), замените в одной из инструкций `FROM` тег на `latest` (и уберите хеш, если он указан)

Перезапустите проверку:

```sh
hadolint -V Dockerfile
```

Убедитесь, что в выводе присутствует предупреждение:

```
Dockerfile:X DL3007 warning: Using latest is prone to errors if the image will ever update. Pin the version explicitly to a release tag
```

И код возврата ненулевой
