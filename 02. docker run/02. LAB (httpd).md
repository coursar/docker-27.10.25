# Apache Server

## Шаг 1. Запуск контейнера

Используя образ [httpd](https://hub.docker.com/_/httpd), запустите Apache Server с учётом следующих требований:
1. Tag: 2.4.65-alpine3.22

2. Port publishing: 8080 хостовой на 80 в контейнере

3. Volume Mapping: htdocs хостовой на /usr/local/apache2/htdocs в контейнере

## Шаг 2. Просмотр приложения

Перейдите в браузере Firefox по адресу http://localhost:8080, скопируйте выведенное на странице значение:

```sh
...fff4
```

Полученное значение и будет ответом

## Шаг 3. Завершение работы контейнера

Завершите работу контейнера с помощью `Ctrl + C` в терминале с контейнером
