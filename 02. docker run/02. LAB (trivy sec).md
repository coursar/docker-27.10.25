# Trivy Secrets

## Шаг 1. Запуск контейнера

Используя образ [trivy](https://hub.docker.com/r/aquasec/trivy), найдите секреты в локальном каталоге `node` хостовой машины.

Команда запуска trivy:

```sh
docker run ->VOLUME MAPPING<- aquasec/trivy fs --scanners secret /node
```

В терминале будет выведена таблица:

```
HIGH: AsymmetricPrivateKey (private-key)
════════════════════════════════════════
Asymmetric Private Key
────────────────────────────────────────
 ???
────────────────────────────────────────
```

Полученное значение и будет ответом

## Шаг 2. Завершение работы контейнера

Убедитесь, что контейнер самостоятельно завершил свою работу (без вашей помощи)
