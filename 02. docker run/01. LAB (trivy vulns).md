# Trivy Vulns

## Шаг 1. Запуск контейнера

Используя образ [trivy](https://hub.docker.com/r/aquasec/trivy), найдите уязвимости в образе Nginx с хешом `@sha256:f547e3d0d5d02f7009737b284abc87d808e4252b42dceea361811e9fc606287f`.

Команда запуска:

```sh
docker run aquasec/trivy image --scanners vuln nginx@sha256:f547e3d0d5d02f7009737b284abc87d808e4252b42dceea361811e9fc606287f
```

В терминале будет выведена таблица:

```
├──────────────────┼─────────────────────┤─────┤
│ nginx            │ CVE-????-????       │ ... │
│                  ├─────────────────────┤─────┤
│                  │ CVE-????-????       │ ... │
├──────────────────┼─────────────────────┤─────┤
```

Полученные значения и будет ответом
